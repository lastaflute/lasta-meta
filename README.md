Lasta Meta
=======================
The engine library to handle metadata of LastaFlute for e.g. LastaDoc, Swagger

# Information
## Maven Dependency in pom.xml
```xml
<dependency>
    <groupId>org.lastaflute.meta</groupId>
    <artifactId>lasta-meta</artifactId>
    <version>0.5.3</version>
</dependency>
```
